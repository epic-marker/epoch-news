 <p>
  【大纪元2023年07月03日讯】（大纪元记者田磊美国圣地亚哥报导）今年7月4日是美国独立247周年纪念日。圣地亚哥县从南到北都有民间组织的
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B8%B8%E8%A1%8C.html">
   游行
  </ok>
  、路跑和街市等
  <ok href="https://www.epochtimes.com/gb/tag/%E5%BA%86%E7%A5%9D.html">
   庆祝
  </ok>
  活动。到了晚上，不少城市的政府和民间团体、商家还会放
  <ok href="https://www.epochtimes.com/gb/tag/%E7%83%9F%E8%8A%B1.html">
   烟花
  </ok>
  。今年还有一些地方因为环保原因，禁放烟花，活动组织者想出了飞行亮灯无人机的替代方案。以下列出部分活动。
 </p>
 <h4>
  圣地亚哥海湾
  <ok href="https://www.epochtimes.com/gb/tag/%E7%83%9F%E8%8A%B1.html">
   烟花
  </ok>
  秀“Big Bay Boom”
 </h4>
 <p>
  时间：7月4日晚9点至9点半
  <br/>
  地点：圣地亚哥港口
  <br/>
  这是美西海岸最大的
  <ok href="https://www.epochtimes.com/gb/tag/%E5%9B%BD%E5%BA%86.html">
   国庆
  </ok>
  烟花秀，由商家运作，在圣地亚哥的Fox5和洛杉矶的KTLA5电视上同步转播。烟花将从圣地亚哥湾内的四艘驳船上燃放。观众可在圣地亚哥海湾的庇护岛（Shelter Island）、海港岛（Harbor Island）、北内河码头、南内河码头、塞萨尔•查韦斯公园（Cesar E. Chavez Park）和科罗纳多渡轮码头等海湾周边地区观看。很多人早早就在海湾沿岸安营扎寨，烧烤，野餐，观看海湾日落，等待晚上的烟花。停车将是一个问题。游客可以乘坐Trolley和渡轮到达目的地，避免停车困扰。
  <br/>
  网站：
  <ok href="https://bigbayboom.com/donate/about-the-fireworks-show/" rel="noopener noreferrer" target="_blank">
   https://bigbayboom.com/donate/about-the-fireworks-show/
  </ok>
 </p>
 <h4>
  科罗纳多市（Coronado）
  <ok href="https://www.epochtimes.com/gb/tag/%E7%8B%AC%E7%AB%8B%E6%97%A5.html">
   独立日
  </ok>
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B8%B8%E8%A1%8C.html">
   游行
  </ok>
  、
  <ok href="https://www.epochtimes.com/gb/tag/%E5%BA%86%E7%A5%9D.html">
   庆祝
  </ok>
  及烟花秀
 </h4>
 <p>
  游行时间：上午10点至中午12点
  <br/>
  游行地点：Orange Ave
  <br/>
  海军降落伞表演：下午2点至3点
  <br/>
  各种音乐会：下午3点至晚9点前
  <br/>
  烟花秀：晚9点至9点半
  <br/>
  烟花秀地点：Glorietta Bay
  <br/>
  科罗纳多位于圣地亚哥县的南端，是海军基地。科罗纳多
  <ok href="https://www.epochtimes.com/gb/tag/%E5%9B%BD%E5%BA%86.html">
   国庆
  </ok>
  游行是圣地亚哥县内规模最大的
  <ok href="https://www.epochtimes.com/gb/tag/%E7%8B%AC%E7%AB%8B%E6%97%A5.html">
   独立日
  </ok>
  游行，今年是第74届。全天的庆祝活动从早上的路跑开始，到游行后的海军跳伞表演，儿童音乐节，爱国音乐节等多个活动，直到晚上的烟花秀。烟花秀与圣地亚哥湾的烟花秀隔湾相望，形成一片，气氛空前。
  <br/>
  网站：
  <ok href="https://coronadofourthofjuly.com/schedule/" rel="noopener noreferrer" target="_blank">
   https://coronadofourthofjuly.com/schedule/
  </ok>
 </p>
 <h4>
  伯纳多庄园（Rancho Bernardo）独立日庆祝和烟花秀
 </h4>
 <p>
  时间：7月4日上午9点开始
  <br/>
  地点：Webb Park公园（11666 Avena Pl）
  <br/>
  庆祝仪式，音乐会，食物，车展，爱国宠物比赛等。
  <br/>
  烟花燃放：晚9点开始
  <br/>
  烟花地点：Bernardo High School（13010 Paseo Lucido）。
  <br/>
  网站：https://spiritofthefourth.org/
 </p>
 <h4>
  波威市（Poway）娱乐和烟花秀
 </h4>
 <p>
  娱乐时间：7月4日上午10点开始
  <br/>
  娱乐地点：老波威公园（Old Poway Park）
  <br/>
  在老波威公园有音乐、儿童手工游戏、食物等娱乐活动。
  <br/>

  <br/>

  <br/>

  <br/>
  烟花秀地点之二：Poway Sportsplex (12349 McIvers Court)
  <br/>
  该烟花地方不能在燃放地点观看，可在周围街道和公园观看。
 </p>

 <p>

  <br/>

 <h4>
  Mira Mesa烟花秀
 </h4>
 <p>
  时间：7月2日晚9点
  <br/>
  地点：Wangenheim Middle School
  <br/>
  娱乐和观看地点：Hourglass Field Community Park社区公园
  <br/>
  这是Mira Mesa和Scripps Mesa地区连续三年停止了国庆烟花秀后第一次重新恢复烟花秀。组织者说，把时间改在7月2日周日，是为了节省开销。这是非营利机构举办，依靠捐款和赞助。
 </p>
 <h4>
  拉荷亚海滩无人机秀
 </h4>
 <p>
  时间：7月4日晚8:45
  <br/>
  地点：Kellogg Park公园
  <br/>
  有着长期传统的拉荷亚海滩（La Jolla Cove）独立日烟花秀，因为环保团体的反对和市议会新条例，连续多年取消。去年圣地亚哥市通过条规，每年5月底至10月底是海狮繁殖的季节，禁止游人靠近拉荷亚海滩一些区域，并禁止在海狮生活的海滩燃放烟花，有效期7年。因此至2029年都不会有烟花秀。
 </p>
 <p>
  但是今年拉荷亚海滩和网球俱乐部将举办一场由100架精心设计的无人机组成的表演。
  <br/>
  表演将于晚上 8:45 开始。 最佳观赏区是Kellogg 公园的南端。 这些无人机由圣地亚哥的无人机工作室控制。#
 </p>
