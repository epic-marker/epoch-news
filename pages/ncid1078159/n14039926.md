 <p>
  【大纪元
  <ok href="https://www.epochtimes.com/gb/tag/2023%E5%B9%B4.html">
   2023年
  </ok>
  07月22日讯】（大纪元记者殷瑞娜编译报导）在本届立法会议上，
  <ok href="https://www.epochtimes.com/gb/tag/%E5%8A%A0%E5%B7%9E.html">
   加州
  </ok>
  众议院议员和参议员提出了数百项
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B3%95%E6%A1%88.html">
   法案
  </ok>
  ，在9月14日午夜之前，他们必须批准或取消这些提案。
 </p>
 <p>
  许多提案会在此过程中被提前否决，如AB 935烟草
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B3%95%E6%A1%88.html">
   法案
  </ok>
  和AB 742警犬法案等。
 </p>
 <p>
  这些法案现已“失效”，意味着本次立法会议不再考虑该法案。但它可能会在未来，作为一项新法案重新提出，或纳入另一项法案。
 </p>
 <p>
  批准的法案将提交给州长纽森，签署成为法律或否决。
 </p>
 <p>
  下面是一些正在参众两院审议的法案：
 </p>
 <h4>
  枪支管控
 </h4>
 <p>
  1. AB 28法案
  <br/>
  该法案将对枪支和弹药销售征收新税，税收资金将用于资助暴力预防项目。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杰西·加布里埃尔（Jesse Gabriel）和另外39位议员。
 </p>
 <p>
  2. SB 2法案
  <br/>
  该法案将加强州对隐藏武器的规定，禁止在包括私人企业在内的许多公共场所携带枪支，除非业主在入口处张贴明显的标志，表明许可证持有者可以携带枪支。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员安东尼·波坦蒂诺（Anthony Portantino ）和另外 24位议员
 </p>
 <p>
  3. SB 377法案
  <br/>
  该法案将禁止警察购买非法手枪，供个人使用或转售。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员南希·斯金纳（Nancy Skinner）
 </p>
 <h4>
  住房
 </h4>
 <p>
  1. AB 1033法案
  <br/>
  该法案将允许城市决定是否允许业主将附属住宅单元（ADU）与主住宅分开出售。 这将废除禁止单独销售附属住宅单元的现行法律。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员丁右立（Philip Ting）。
 </p>
 <p>
  2. AB 1114法案
  <br/>
  该法案将将阻止城市允许建筑许可上诉， 即使申请人已经获得开发土地的权利。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）。
 </p>
 <p>
  3. AB 12法案
  <br/>
  该法案将禁止房东，向租户收取超过一个月的租金作为保证金。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）和另外13位议员。
 </p>
 <p>
  4. AB 1485法案
  <br/>
  该法案将赋予
  <ok href="https://www.epochtimes.com/gb/tag/%E5%8A%A0%E5%B7%9E.html">
   加州
  </ok>
  总检察长，无条件介入任何因可能违反州住房法而提起的诉讼的权利。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）和另外一位议员。
 </p>
 <p>
  5. AB 1532法案
  <br/>
  该法案将加快将空置办公楼改造成住房的项目。 无论分区如何，所有地区都将允许此类项目。
  <br/>
  该法案尚在众议院委员会审议中。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）
 </p>
 <h4>
  AB 572法案
 </h4>
 <p>
  对于经济适用住房（即低于市场价格的单位）的业主，物业管理费用（HOA）的增幅，将限制在每年5%。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）
 </p>
 <p>
  6. AB 68法案
  <br/>
  该法案将大大增加在野火或洪水易发地区建造小区的难度。与此同时，它还要求城市让开发商更容易、更便宜地在城市地区建造住房。
  <br/>
  该法案尚在众议院委员会审议中。
  <br/>
  撰稿：民主党议员克里斯·沃德（ Chris Ward）
 </p>
 <p>
  7. AB 966法案
  <br/>
  该法案将永久性地禁止各城市通过法令，要求业主在其附属住宅单元（ADU） 中居住。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：共和党议员劳里·戴维斯（Laurie Davies）
 </p>
 <p>
  8. SB 4法案
  <br/>
  该法案将允许宗教团体，有权在其额外土地上建造经济适用房，消除当地分区限制或酌情审批程序造成的障碍。它将适用于教堂、犹太教堂、清真寺和其它信仰组织，以及非营利性大学。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员威善高（Scott Wiener）和另外15位议员。
 </p>
 <p>
  9. SB 423法案
  <br/>
  将永久延长2017年颁布的SB 35法案，该法律将于 2025年年底到期。如果城市未达到州规定的住房目标，则迫使地方政府简化某些项目的审批。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员威善高（Scott Wiener）和另外3位议员。
 </p>
 <h4>
  无家可归
 </h4>
 <p>
  1. AB 464法案
  <br/>
  该法案将让无家可归者和低收入者，免费获得新身份证，并复制出生证明等重要记录。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员皮拉尔·夏沃（Pilar Schiavo ）和另外10位议员。
 </p>
 <p>
  2. SB 363法案
  <br/>
  该法案将创建一个行为健康治疗床位的数据库。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员苏珊·艾格曼（Susan Eggman）和另外3位议员。
 </p>
 <p>
  3. SB 43法案
  <br/>
  该法案将使患有严重精神疾病的人更容易接受监管
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员苏珊·艾格曼（Susan Eggman）和另外22位议员。
 </p>
 <h4>
  犯罪
 </h4>
 <p>
  1. AB 37法案
  <br/>
  该法案允许政治家将竞选资金用于保安，包括为自己或家人提供保镖或家庭保安系统。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员米娅·邦塔（Mia Bonta）和另外1位议员。
 </p>
 <p>
  2. AB 452法案
  <br/>
  该法案将取消 2024 年 1 月 1 日之后发生的儿童性侵犯民事诉讼的诉讼时效。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员道恩·阿迪斯（Dawn Addis ）和另外19位议员。
 </p>
 <p>
  3. AB 474法案
  <br/>
  该法案将要求州应急官员和警察与当地执法部门合作，打击芬太尼交易。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员弗雷迪·罗德里格斯（Freddie Rodriguez ）和另外1位议员。
 </p>
 <p>
  4. AB 701法案
  <br/>
  该法案将增加对芬太尼交易罪名成立的人的刑期和罚款。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员科蒂·皮特里-诺里斯（Cottie Petrie-Norris） 和另外14位议员。
 </p>
 <p>
  5. AB 881法案
  <br/>
  该法案将增加陪审团中低收入陪审员的薪酬。刑事案件中，工资不超过所在县平均工资80%，且雇主不支付休息时间工资的陪审员，每天可获得100美元的补助。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员丁右立（Philip Ting）
 </p>
 <p>
  6. ACA4法案
  <br/>
  该法案将修改州宪法，允许被监禁的加州人在监狱中投票。
  <br/>
  该法案正在众议院审议。
  <br/>
  撰稿：民主党议员艾萨克·布莱恩（Isaac Bryan）和另外15位议员。
 </p>
 <p>
  7. SB 50法案
  <br/>
  该法案将限制警察进行拦截的能力，即警察因轻微交通违规（如尾灯损坏、登记过期或有色车窗）而拦截司机，并以拦截为借口，调查司机是否有更严重的违法行为。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员史蒂文·布拉德福德（Steven Bradford）和另外9位议员。
 </p>
 <p>
  8. SB 58法案
  <br/>
  该法案将使一些致幻药物的使用合法化，例如致幻蘑菇和DMT（二甲基色胺）。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员威善高（Scott Wiener）和另外14位议员。
 </p>
 <h4>
  环境
 </h4>
 <p>
  1. AB 1347法案
  <br/>
  该法案要求企业仅在顾客要求时打印收据。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员丁右立（Philip Ting）
 </p>
 <p>
  2. AB 591法案
  <br/>
  该法案将要求加州所有新建和改造的电动汽车充电站，向所有类型的电动汽车开放并可使用。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杰西·加布里埃尔（Jesse Gabriel）和另外1位议员。
 </p>
 <p>
  3. AB 660法案
  <br/>
  该法案将要求食品制造商在保质期标签上，使用相同的短语：“最佳使用期限”表示最佳新鲜度或质量，“使用期限”表示食品安全。该法案还将禁止销售截止日期和许多其它没有普遍意义的标签。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员雅基·欧文（Jacqui Irwin）
 </p>
 <p>
  4. SB 252法案
  <br/>
  该法案将要求美国最大的两个公共养老基金—加州公共雇员退休系统(CalPERS)和加州教师退休系统(CalSTRS)，撤出其在化石燃料公司持有的110亿美元资金。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员莉娜·冈萨雷斯（Lena Gonzalez）和另外3位议员。
 </p>
 <p>
  5. SBX 12法案
  <br/>
  这项法案将赋予加州能源委员会权力，对炼油商的利润设定上限，并设定数额。他们还有权对超过上限的公司进行罚款，并设定罚款金额。
  <br/>
  该法案已签署成为法律。
  <br/>
  撰稿：民主党议员南希·斯金纳（Nancy Skinner）
 </p>
 <h4>
  税务
 </h4>
 <p>
  1. AB 259法案
  <br/>
  该法案将对加州最富有的人的总资产征收财富税，而不仅仅是他们去年赚到的新钱。
  <br/>
  该法案尚在众议院委员会审议。
  <br/>
  撰稿：民主党议员亚历克斯·李（Alex Lee）和其他9位议员。
 </p>
 <p>
  2. ACA3法案
  <br/>
  这是一项拟议的宪法修正案，允许州征收财富税，并取消20世纪70年代加州选民通过的州收入上限，即江恩限额（Gann Limit）。
  <br/>
  该法案尚在众议院委员会审议。
  <br/>
  撰稿：民主党议员亚历克斯·李（Alex Lee）和其他9位议员。
 </p>
 <h4>
  健康
 </h4>
 <p>

  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员莉兹·奥尔特加（Liz Ortega）和其他7位议员。
 </p>
 <p>
  2. AB 663法案
  <br/>

  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）
 </p>
 <p>
  3. SB 363法案
  <br/>
  该法案将创建一个行为健康治疗床位数据库。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员苏珊·艾格曼（Susan Eggman）和其他3位议员。
 </p>
 <p>
  4. SB 770法案
  <br/>

  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员迈克·麦奎尔（Mike McGuire）和其他8位议员。
 </p>
 <p>

  <br/>

  <br/>

  <br/>
  撰稿：民主党议员威善高（Scott Wiener）和其他4位议员。
 </p>
 <h4>
  工作场所
 </h4>
 <p>
  1. AB 1法案
  <br/>
  该法案将允许州立法者的工作人员组建工会。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员艾萨克·布莱恩（Isaac Bryan）和其他42位议员。
 </p>
 <p>
  2. AB 1228法案
  <br/>
  该法案将使企业快餐店特许经营者，对特许经营地点的工作场所违规行为承担责任。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员克里斯·霍尔登（Chris Holden）
 </p>
 <p>
  3. AB 1356法案
  <br/>
  该法案将扩大州法律要求解雇通知或遣散费的范围，将合同工纳入其中，并将要求从60天延长至90天。它还将禁止公司强迫工人签署保密协议以获得通知或工资。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）和其他7位议员
 </p>
 <p>
  4. AB 316法案
  <br/>
  该法案将要求公司测试半挂卡车自动驾驶技术时，必须有一名人类操作员在场。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员塞西莉亚·阿吉亚尔-库里（Cecilia Aguiar-Curry）和其他27位议员
 </p>
 <h4>
  选举
 </h4>
 <p>
  1. SCA 1法案
  <br/>
  它将在2024年的选票上提出一项措施，要求选民批准一项宪法修正案，通过取消选票上的第二个问题，即选择可能的替代者，对加州的罢免选举程序进行全面改革。
  <br/>
  该法案正在众议院审议。
  <br/>
  撰稿：民主党议员乔什·纽曼（Josh Newman）和其他4位议员。
 </p>
 <h4>
  毒品和酒精
 </h4>
 <p>
  1. AB 374法案
  <br/>
  该法案将允许加州政府，允许大麻企业提供食品和非酒精饮料，并举办现场音乐表演。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员杨驰马（Matt Haney）
 </p>
 <p>
  2. SB 277法案
  <br/>
  该法案将允许杂货店出售罐装鸡尾酒。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员比尔·多德（Bill Dodd）
 </p>
 <h4>
  社会公平
 </h4>
 <p>
  1. ACA 5法案
  <br/>
  该法案将把婚姻平等权利写入州宪法，并从州宪法中废除8号提案，该提案于2008年选民通过，禁止同性婚姻。
  <br/>
  该法案已在众议院和参议院获得通过。
  <br/>
  撰稿：民主党议员埃文·洛（Evan Low）和其他77位议员
 </p>
 <p>
  2. SB 447法案
  <br/>
  该法案将废除加州禁止州资助前往23个颁布歧视LGBTQ人群政策的州旅行的法律。
  <br/>
  该法案已在参议院获得通过，目前正在众议院审议。
  <br/>
  撰稿：民主党议员托妮·阿特金斯（Toni Atkins）
 </p>
 <h4>
  堕胎
 </h4>
 <p>
  AB 793法案
  <br/>
  该法案将禁止反向定位和关键词搜查令，这些搜查允许警方锁定访问过特定地点（例如堕胎诊所），或搜索过某些关键词的人。
  <br/>
  该法案已在众议院获得通过，目前正在参议院审议。
  <br/>
  撰稿：民主党议员米娅·邦塔（Mia Bonta）和其他7位议员。
 </p>
