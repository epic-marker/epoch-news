   <ok href="https://www.epochtimes.com/gb/tag/%E6%9C%BA%E5%AF%86%E6%96%87%E4%BB%B6%E6%A1%88.html">

  </span>
  <span class="s2">

  </span>
  <span class="s1">
   。
  </span>
 </p>

  <span class="s3">
   7月27日，史密斯再就此案对川普追加3项指控。
  </span>
  <span class="s1">
   在起诉书中，检方表示，美国司法部去年
  </span>
  <span class="s4">
   5
  </span>
  <span class="s1">
   月发出传票、要求川普归还机密文件后，川普为湮灭证据，要求员工德‧奥利维拉（
  </span>
  <span class="s4">
   Carlos De Oliveira
  </span>
  <span class="s1">

  <span class="s4">

  <span class="s4">
   Mar-a-Lago
  </span>
  <span class="s1">
   ）的监控录像。

 </p>
 <p class="p6">
  <span class="s1">
   德‧奥利维拉也被控向美国联邦调查局（
  </span>
  <span class="s4">
   FBI
  </span>
  <span class="s1">
   ）调查探员说谎，谎称

  <span class="s1">

   31
  </span>
  <span class="s1">

   沃尔特

  <span class="s7">

  </span>
 </p>
 <p class="p6">
  <span class="s1">

 <p class="p10">

  <span class="s1">
   川普和诺塔此前已经出席最早的起诉传讯，因此他们有权在新指控文件中放弃亲自出庭受审的权利。
  </span>
 </p>
 <p class="p10">
  <span class="s1">
   川普周五向联邦法院提交的一份文件中提出了他的抗辩和弃权。
  </span>
 </p>
 <p class="p10">
  到目前为止，诺塔尚未提交法庭文件进行抗辩，也没有放弃对新增指控亲自出庭的权利。
 </p>
 <p class="p11">
  <span class="s1">
   联邦法官已安排2024年5月20日开始审判此案。
  </span>
 </p>
 <p class="p10">
  <span class="s1">
   周四（3日），川普对华盛顿特区联邦法院审理的另一起案件中也表示不认罪，他被指控犯有四项与他力求推翻2020年选举结果有关的重罪。
  </span>
 </p>
 <p class="p12">
  <span class="s1">
   今年早些时候，纽约州地区检察官白艾荣（Alvin Bragg）在州一级提起对川普涉嫌向艳星斯托米‧丹尼尔斯（Stormy Daniels）支付“封口费”一案提出34项财务相关的指控。
  </span>
 </p>
 <p class="p12">
  <span class="s1">
   川普对所有指控均不认罪，并坚称他是因政治原因而遭起诉，因为他是2024年共和党总统竞选的领跑者。
  </span>
 </p>
 <p class="p12">
