  赤鼠时同运不同

  中原好景不为功
  <br/>
  西方再见南军至
  <br/>
  刚到金蛇运已终
  <br/>
  这是黄檗禅师预言中共的一首诗。黄檗禅师的预言诗共14首，前边12首从明亡到民国、两岸分治已经全部应验。这是第13首，关于中共，最后一首是结语，没有实质内容。
  <br/>
  这首诗第一句提到鼠，2020年为鼠年。最后一句提到蛇，2025年为蛇年。第二句好理解，“中原好景不为功”，中国的经济发展也算不上它的功劳。关键是第三句“西方再见南军至”，西方对应五脏为肺，南方对应五行为火，再见为双火，所以这句话可以理解为“肺炎”。
  <br/>
  这样看这首诗的意思就很清楚了，中共将在一场“肺炎”瘟疫中结束，这场瘟疫开始的时间是鼠年（2020年），结束的时间是蛇年（2025年）。
  <br/>
  从目前来看，应验的可能性也很大，至少前边三句已经应验了。
  <br/>
  毛泽东说，他是“和尚打伞—无法无天”，但是8341的故事证明其一生从生到死都已注定。
  <br/>
  中共张牙舞爪，战天斗地，同样生死有命。造出它这么个坏东西就是利用它干坏事的，利用完了它也就完了。所有跟着中共跑的或者被它欺骗利用的都将成为其陪葬，去跟它一起偿还永远也还不完的罪孽。
  <br/>
  我们声明退出中共这个魔鬼组织，跟着魔鬼走只有作恶犯罪下地狱，不会有第二种结果。
