 <p>
  【大纪元2023年10月12日讯】（大纪元记者骆亚采访报导）近来中共高层乱象纷呈，尤其是国防部长
  <ok href="https://www.epochtimes.com/gb/tag/%E6%9D%8E%E5%B0%9A%E7%A6%8F.html">
   李尚福
  </ok>
  、外交部长
  <ok href="https://www.epochtimes.com/gb/tag/%E7%A7%A6%E5%88%9A.html">
   秦刚
  </ok>
  至今无下落，军中高级将领大批遭整肃，成为国际舆论焦点。
 </p>
 <p class="p1">
  <span class="s1">
   香港著名作家、评论家
   <ok href="https://www.epochtimes.com/gb/tag/%E9%A2%9C%E7%BA%AF%E9%92%A9.html">
    颜纯钩
   </ok>
  </span>
  <span class="s2">
   10
  </span>
  <span class="s1">
   月
  </span>
  <span class="s2">
   12
  </span>
  <span class="s1">
   日接受大纪元专访，分析了习近平第三任期下中南海的风暴，正在发生的灰犀牛事件和可能的黑天鹅事件。下面是专访纪实。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：习近平的外交部长
    <ok href="https://www.epochtimes.com/gb/tag/%E7%A7%A6%E5%88%9A.html">
     秦刚
    </ok>
    和国防部长
    <ok href="https://www.epochtimes.com/gb/tag/%E6%9D%8E%E5%B0%9A%E7%A6%8F.html">
     李尚福
    </ok>
    目前失踪，您怎么看习近平自己钦定的这两个部长的遭遇？这是否是中共宫廷斗争的必然结果？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   中共最高层政治内斗，从建政第一天起就没有停过。毛泽东亲手干掉的接班人，包括刘少奇﹑林彪，邓小平接手干掉的，有胡耀邦﹑赵紫阳。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   习近平上台后，以反腐之名干掉更多。秦刚与李尚福，当然都是权争的牺牲品，他们已是过去式的人物，在中共体制下，这是常态，往后要关心的，是还有谁会中招。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   习近平已是典型的孤家宾人，极度缺乏安全感，他上台后，花最大心思在整肃异己，现在与毛泽东晚年同样遭遇，便是百年后将受历史诅咒的命运，所以不惜以最大代价，来巩固自己的班底，维护自己的地位。因此，类似的清洗还会不断发生，到最后，只剩眼前一小撮马屁精，和他最后面对笈笈可危的中共江山。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：您觉得习近平现在大规模整顿军队有哪几个主要原因？与台海危局关系如何？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   说大规模整顿军队言之尚早，目前只是整顿火箭军，其它野战军都没有动。火箭军是否牵涉到贪腐问题，或者在采购军备上出了什么大纰漏，目前也不清楚，还需要观察。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   整顿军队与台海危机关系不大，主要是习近平危机感太重，因为处处感到受威胁，所以疑神疑鬼，所以要拆除地雷，但究竟有没有那个地雷，恐怕他自己也不知道。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   有些事是应该做的，有些事只是以为应该做，实际是不应该做，或根本不能做的。但人在失去常性之下，最容易做误判，皇帝不信任官吏，动辄杀人，往往把精英都杀光，只剩下一帮没用的奴才，这是最终断送自己必经之路。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：习近平的第三任期，他现在面临的最大的黑天鹅和灰犀牛事件是什么？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   习近平第三任期，实际上都在承受前两个任期留下来的政治负资产。目前摆在他面前的几只灰犀牛，首先是中美关系恶化，已经不可逆转，往后只是更深化与扩大的问题，而对此，习近平完全无能为力，只能被动接受，尽可能减少伤害；
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   第二只灰犀牛是大陆经济沦落，这件事他也基本没有解方；
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   第三只灰犀牛是港台，香港国安法之下全面败坏，台湾文统武统之路也都走不通，港台都是他要吞下去的苦果；
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   第四只灰犀牛是欧盟、日本、印度、澳洲，甚至菲律宾、越南等东南亚一些小国，都与中共离心离德，习近平放眼世界，只剩俄罗斯普京一个朋友，以及那些靠中共接济的小猫三四只，在国际关系上彻底孤立，处境只会越来越艰难。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   至于黑天鹅，本来就不可预测，但黑天鹅来与不来，也都有前因后果，不会平白无故飞一只黑天鹅来。最近开始发生个别小银行挤提的事，这只是一种苗头，但若有大型银行发生类似事件，那就是黑天鹅了。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   中共为解决眼下困境，不断推出救急措施，但因习近平已失方寸，随时可能出台一些激怒民意的政策，那也很可能是黑天鹅。中共飞机舰艇频繁犯台，或许有一天发生擦枪走火事件，由此引发台海战争，最终美日韩卷入，那也是黑天鹅。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：习近平现在强调的斗争、甚至搬出枫桥经验，这是中共政权走入困境压制民间的举措，这会起到预期的效果吗？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   关于斗争，有两个涵义，一是对外斗争，二是对内斗争。对内斗争又有两个层面，一是党内不同路线的斗争，一是对民间反抗的镇压。因为形势恶劣，预后凶险，习近平不得不做最坏的打算。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   最坏的打算便是民变四起，地方政府失能，中共唯有动用武装力量镇压，所以最近又到处成立武装部，以便灵活应对社会的动乱。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   至于枫桥经验，只是群众斗群众的方式，挑动不明真相的群众，帮忙中共政权对付民间反抗力量，这只是为灭轻政府压力的一种补救措施。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   但当整个社会陷入万劫不复之境地，还有多少群众会听从中共召唤，到民变现场去为中共维稳，那还是一个疑问。所以枫桥经验只是习近平在束手无策之下，勉强找到的一根救命稻草，对巩固政权没什么实质帮助，真正能起镇压作用的，还是武装。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：习近平在中共政治局第八次集体学习时强调积极参与世界贸易组织改革，提高驾驭高水平对外开放能力，您怎么看？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   习近平永远在做他“中国解决方案”的大梦，“一带一路”，海外文宣，渗透联合国机构，大撒币收买中小国家，这些都是他经营世界格局的大想头。他的进取表现在战狼外交，却得罪了全世界。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   现在美欧已全面反省，结成反共统一阵线，习近平经营世界的那支歌子已经唱不响了。他面临的，不是参与世贸组织改革，他面临的是美欧等国改革联合国体制，扶持台湾加入联合国属下机构，甚至西方民主国家承认台湾独立主权地位的威胁。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   习近平喜欢讲大话，搞宏大叙事，以显示自己千古一帝的历史地位，这是他不自量力的表现。有经世之才，先搞好眼下的中国经济，中国经济死，还有什么活得下去，改革世贸只是一场春梦。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：您怎么看习近平现在跟中共官场官员、跟中共的元老、红二代、太子党的关系？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   习近平与红二代是命运共同体，与中共元老也是。但最失败的是，习近平没有把中共江山打理好，反倒败坏了权贵集团的根本利益，因此元老与红二代中想要推倒习近平的大有人在。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   问题是，推倒习近平，找谁去代替他？遍观中共官场，现在有能力有声望镇得住场面，可以起死回生的人，一个都没有。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   与其换人而换来更大风险，不如勉强维持现状，施加压力给习近平，那样可能还苟延残喘更多时间。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   现在明白中共江山不保的红二代与老人帮一定不少，实际上他们也没有“救国（党）良策”，他们只是在一旁发发牢骚而己。船一沉大家都落水，只是不要那么快沉，保得住多少保多少。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    记者：您怎么预估中南海的未来走向，及中国社会未来的走向？
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   中南海未来走向，就是风雨飘摇中苟延残喘下去，泰山压顶之灾避不过，问题只是能挺多长时间而已。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   中国社会的未来，一定会经过一个政权崩溃的过程，快则五年慢则十年，政权解体，全国陷于大乱；因为民间恶质化，要重头收拾旧河山将是艰难的历史工程。河山澄清﹑普世价值统一中国的日子，目前看不到。
  </span>
 </p>
