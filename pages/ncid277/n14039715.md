 <p>
  【大纪元2023年07月22日讯】（大纪元记者程静报导）在中共领导人
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B9%A0%E8%BF%91%E5%B9%B3.html">
   习近平
  </ok>
  加强控制中国大学教育，清洗自由派思想家，推动大学“
  <ok href="https://www.epochtimes.com/gb/tag/%E5%85%9A%E5%BB%BA.html">
   党建
  </ok>
  ”之际，西方学术机构与中国的合资大学还在扩张，并在当局的压力下坚持“学术自由”。
 </p>
 <h4 class="p1">
  中共控制大学教育
 </h4>
 <p class="p1">
  <span class="s1">
   <span class="Apple-converted-space">
    《经济学人》日前报导，
    <span class="s2">
     当纽约大学（
    </span>
    NYU
    <span class="s2">
     ）上海分校毕业典礼奏起典型美国风格的
    </span>
   </span>
  </span>
  《威风凛凛进行曲》（Pomp and Circumstance March Number 1）时，毕业生们可能反思西方文化移植中国的奇怪之处，不仅是音乐，还有他们作为本科生的整个经历。
 </p>
 <p class="p1">
  <span class="s2">
   近年来，中共与西方特别是美国的关系日益紧张。
  </span>
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B9%A0%E8%BF%91%E5%B9%B3.html">
   习近平
  </ok>
  <span class="s2">
   正试图清除大学里的自由主义思想。然而，这里有近
   <span class="s1">
    400
   </span>
   名学生，其中约一半是中国人，从美国学术机构的中国分校获得学士学位证书。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   自习近平
  </span>
  <span class="s2">
   2012
  </span>
  <span class="s1">
   年上台后，加强了对大学教育的控制，
  </span>
  <span class="s1">
   一些自由派学者被解雇，
  </span>
  教
  <span class="s1">
   师被勒令停止使用被认为宣扬西方价值观的进口教科书，
  </span>
  <span class="s1">
   大学被告知要成为
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   共产党领导的堡垒
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   在习近平的领导下，很少有学生敢于举行抗议活动。
  </span>
  <span class="s1">
   但去年
  </span>
  <span class="s2">
   11
  </span>
  <span class="s1">
   月，多个城市的校园爆发了小规模示威活动，抗议与新冠病毒相关的严厉封锁。
  </span>
  中共
  <span class="s1">
   评论员指责
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   外国势力
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   煽动骚乱。
  </span>
  <span class="s1">
   这无疑使北京对西方的影响更加警惕。
  </span>
 </p>
 <h4>
  <ok href="https://www.epochtimes.com/gb/tag/%E7%BE%8E%E8%8B%B1%E5%A4%A7%E5%AD%A6%E4%B8%AD%E5%9B%BD%E5%88%86%E6%A0%A1.html">
   美英大学中国分校
  </ok>
  办学规模扩大
 </h4>
 <p class="p1">
  <span class="s1">
   不过，像上海纽约大学这样的企业气氛似乎不太一样，
  </span>
  <span class="s1">
   甚至规模还在扩大。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   英国诺丁汉大学于
  </span>
  <span class="s2">
   2004
  </span>
  <span class="s1">
   年在浙江宁波建立了校区以后，
  </span>
  <span class="s1">
   上海纽约大学和昆山杜克大学也在中国建立了分校。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   今年，纽约大学位于上海浦东金融区的校园，搬到了一座由国际著名建筑事务所
  </span>
  <span class="s2">
   Kohn Pedersen Fox
  </span>
  <span class="s1">
   设计的耀眼的新九层建筑，教室空间是上海纽约大学原址的两倍。这家总部位于纽约的建筑公司曾设计浦东最具标志性的摩天大楼
  </span>
  <span class="s2">
   ——
  </span>
  <span class="s1">
   上海环球金融中心。
  </span>
 </p>
 <p class="p1">
  <span class="s2">
   8
  </span>
  <span class="s1">
   月，位于上海附近的昆山杜克大学
  </span>
  <span class="s2">
   (DKU)
  </span>
  <span class="s1">
   也将迁入新址。
  </span>
  <span class="s1">
   该合资大学由北卡罗来纳州杜克大学、武汉大学和昆山市政府共同举办，占地超过
  </span>
  <span class="s2">
   33
  </span>
  <span class="s1">
   公顷，是现有校园的两倍。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   在附近的城市苏州，由西安交通大学和英国利物浦大学合资的西交利物浦大学，拥有
  </span>
  <span class="s2">
   23,000
  </span>
  <span class="s1">
   名学生，是中国最大的学术合资企业。
  </span>
  <span class="s1">
   去年，它又开设了一个校园，可容纳
  </span>
  <span class="s2">
   10,000
  </span>
  <span class="s1">
   人。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   报导说，尽管习近平日益反对西方，此类企业仍不断涌现，虽然规模较小。
  </span>
  <span class="s1">
   总的来说，中国逾
  </span>
  <span class="s2">
   1,230
  </span>
  <span class="s1">
   所大学中有超过一半与外国或香港同行有合作项目。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   在过去的十年中，此类计划的数量（其中大多数涉及英语国家的大学）增加了三分之一。
  </span>
  <span class="s1">
   它们的范围从十所成熟的大学到众多小型机构，例如中国校园内的联合院系或研究所。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   不过，与中国其它大学系统的招生规模相比，外国办学项目的学生人数很少：据官方媒体报导，不到
  </span>
  <span class="s2">
   60
  </span>
  <span class="s1">
   万人，即不到中国学生总数的
  </span>
  <span class="s2">
   2%
  </span>
  <span class="s1">
   。
  </span>
 </p>
 <h4>
  习近平要大学加强
  <ok href="https://www.epochtimes.com/gb/tag/%E5%85%9A%E5%BB%BA.html">
   党建
  </ok>
 </h4>
 <p>
  中共担心西方思想的潜在传播，2017年发布了在外国学术项目中实施中央关于“党建”指示的指导方针，要求
  <span class="s2">
   该合资企业的党委书记（中国合作伙伴总是有一名党委书记）担任董事会成员。
  </span>
 </p>
 <p class="p1">
  <span class="s2">
   然而，在昆山杜克大学，党委书记没有投票权。
  </span>
  <span class="s2">
   时任昆山杜克大学执行副校长的丹尼斯‧
  </span>
  <span class="s2">
   西蒙
  </span>
  <span class="s1">
   (Denis Simon)
  </span>
  <span class="s2">
   表示，允许其投票可能会改变美中代表之间的平衡。
  </span>
  <span class="s2">
   他说，这需要重写合资协议。
  </span>
  <span class="s2">
   美国人并不热衷于这样做。
  </span>
  <span class="s2">
   这个问题
  </span>
  <span class="s1">
   “
  </span>
  <span class="s2">
   从未完全解决
  </span>
  <span class="s1">
   ”
  </span>
  <span class="s2">
   。
  </span>
 </p>
 <p class="p1">
  <span class="s2">
   报导说，中共教育机构变得焦躁不安。
  </span>
  <span class="s1">
   2020
  </span>
  <span class="s2">
   年重庆教育学院（拥有众多国外合作伙伴）
  </span>
  <span class="s2">
   指示，中外合资的中方师生要
  </span>
  <span class="s1">
   “
  </span>
  <span class="s2">
   自觉抵制西方各种错误思潮和腐朽思想的腐蚀
  </span>
  <span class="s1">
   ”，
  </span>
  <span class="s2">
   用
   <span class="s1">
    “
   </span>
   督导员
   <span class="s1">
    ”
   </span>
   来监督学生。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   2021
  </span>
  <span class="s2">
   年，利物浦大学的一名党内官员说，党委应确保不任命任何有
  </span>
  <span class="s1">
   “
  </span>
  <span class="s2">
   意识形态问题
  </span>
  <span class="s1">
   ”
  </span>
  <span class="s2">
   的人担任教员。
  </span>
  <span class="s2">
   宁波诺丁汉大学党委书记说：
  </span>
  <span class="s1">
   “
  </span>
  <span class="s2">
   如果学校在意识形态等重要问题上犯了错误，那么学校就没有存在的价值，需要关闭。
  </span>
  <span class="s1">
   ”
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   今年3月，当局表示，在海南岛专门从事工程和医学等领域的外国大学可以自己设立校园，而现有企业则必须有中国合作伙伴。
  </span>
  但
  <span class="s1">
   董事会中仍需要一名党委书记。
  </span>
 </p>
 <h4 class="p1">
  西方职员如何坚持“学术自由”
 </h4>
 <p>
  <span class="s1">
   2018
  </span>
  <span class="s2">
   年，时任宁波诺丁汉大学教授的斯蒂芬‧
  </span>
  <span class="s2">
   摩根（现已退休）表示，他被迫辞去董事会职务。
  </span>
  <span class="s2">
   他指责共产党：他在网上写了一篇批评习近平的文章，
  </span>
  <span class="s1">
   “
  </span>
  <span class="s2">
   至少一年来，很多党员甚至不看我一眼，也不向我打招呼。
  </span>
  <span class="s1">
   ”
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   不过，上海纽约大学副校长杰弗里‧
  </span>
  <span class="s1">
   雷曼
  </span>
  <span class="s2">
   (Jeffrey Lehman)
  </span>
  <span class="s1">
   仍然保持着坚忍不拔的态度。
  </span>
 </p>
 <p class="p1">
  <span class="s2">
   “
  </span>
  <span class="s1">
   在我们的学术界，学术自由得到了充分尊重。
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   他说，
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   学术自由从一开始就是明确的认识。
  </span>
  <span class="s1">
   如果中国政府对此不再满意，我们就可以关闭。
  </span>
  <span class="s2">
   ”
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   利物浦大学在回应中共官员的言论时表示，该校
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   绝对致力于学术独立
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   ，该合资企业
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   将言论自由视为其核心价值观
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   尽管北京忧心忡忡，但并没有表现出想要赶走已经建立了高影响力的著名大学品牌的迹象。
  </span>
  <span class="s1">
   但摩根先生认为，涉及人文或社会科学的新项目不太可能获得批准。
  </span>
 </p>
