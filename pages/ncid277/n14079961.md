 <p>
  【大纪元2023年09月24日讯】（大纪元记者王若曦采访报导）
  <span class="s2">
   原
  </span>
  <span class="s1">
   沈阳市公安局局长
   <ok href="https://www.epochtimes.com/gb/tag/%E8%AE%B8%E6%96%87%E6%9C%89.html">
    许文有
   </ok>
   被查的消息做实后，许文有的老上级、曾任沈阳市市委书记的曾维也传出被中纪委带走。知情人称，曾维在沈阳市经营了多年，许文有是他的打手。
  </span>
 </p>
 <p class="p5">
  <span class="s3">
   日前知情人告诉大纪元记者，北京时间周三（9月20日），
  </span>
  <span class="s4">
   许文友的原上级曾维被中纪委带走了。
  </span>
  <span class="s1">
   曾维是原沈阳市市委书记，省委常委，原辽宁省委
   <ok href="https://www.epochtimes.com/gb/tag/%E7%A7%98%E4%B9%A6%E9%95%BF.html">
    秘书长
   </ok>
   ，李中堂（李克强）的大秘书。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   上个月（
  </span>
  <span class="s5">
   8月21日
  </span>
  <span class="s1">
   ），原沈阳市公安局局长
   <ok href="https://www.epochtimes.com/gb/tag/%E8%AE%B8%E6%96%87%E6%9C%89.html">
    许文有
   </ok>
   被纪委带走的消息经多方传出。知情人称，许文有的案子辽宁办不了，许文有被带到外地调查。去年他已经进去了一次了，但是半年就给放了，他把纪委都给买通了。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   知情人认为，这个案子很大的，不然许文有一个厅局级干部、副市长，不会把他带到外地去调查。许文友案是涉及到沈阳市委书记曾伟，李克强在辽宁当书记的时候，曾维是
   <ok href="https://www.epochtimes.com/gb/tag/%E7%A7%98%E4%B9%A6%E9%95%BF.html">
    秘书长
   </ok>
   。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   公开资料显示，曾维生于1956年12月，2002年任盘锦市委书记；2004年任辽宁省委秘书长、辽宁省委常委、省委办公厅主任，省直机关工委书记；2008年1月任沈阳市委书记；2015年7月升任辽宁省委副书记兼沈阳市委书记。
  </span>
 </p>
 <p class="p8">
  <span class="s1">
   从履历上看，曾维和许文有在沈阳的官场交集高度重合。
  </span>
 </p>
 <p class="p9">
  <span class="s6">
   2016年8月，
  </span>
  <span class="s1">
   辽沈晚报消息，厦门市委书记王蒙徽调任沈阳书记，曾维不再担任。
  </span>
  <span class="s7">
   2018年1月，曾维不再担任辽宁省人大常务委员会副主任。
  </span>
 </p>
 <p class="p11">
  <span class="s1">
   知情人称，十年前他们（纪委）就要查曾维，2013年
   <ok href="https://www.epochtimes.com/gb/tag/%E8%BE%BD%E5%AE%81%E8%B4%BF%E9%80%89%E6%A1%88.html">
    辽宁贿选案
   </ok>
   的时候，就有人告发他，后来就不了了之了。因为习近平刚上来，要整江派“辽宁帮”的王岷。
  </span>
 </p>
 <p class="p11">
  <span class="s1">
   知情人还透露，许文有卖官不收钱，他是卖权，拿权力换权力。比如，一个派出所副所长，他能把他从一个副科提副处，直接就跳级，只要上面人打招呼。
  </span>
 </p>
 <p class="p12">
  <span class="s1">
   “许文有当公安局长的时候，谁都害怕他。他是个工作狂，七点钟上班开会了，警察六点半就得到了，各区的局长许文有训他们就像训孙子似的，可得瑟了。因为后面是曾维，市委书记支持他，他说撤谁就撤谁。”
  </span>
 </p>
 <p class="p12">
  <span class="s1">
   “曾维在沈阳市也挺霸道。谁要是对他不满了，谁要是告他了，那他就让许文有出头处理，许文有帮他维稳嘛。”
  </span>
 </p>
 <p class="p11">
  <span class="s1">
   “一动许文有，后面就是曾维，曾维后面就是李中堂了。我估计人家来弄他，肯定就定下来了，轻易不能带走。”
  </span>
 </p>
 <p class="p11">
  <span class="s1">
   知情人回顾，李克强任辽宁省省委书记的时候，铁岭有一个钢厂发生
   <ok href="https://www.epochtimes.com/gb/tag/%E9%92%A2%E6%B0%B4%E5%8C%85%E5%80%BE%E8%A6%86.html">
    钢水包倾覆
   </ok>
   特别重大事故，逾30人（引用官媒数据仅供参考）被钢水熔化（钢水超过1500度）了。因为中央要提拔李克强，这事就压下了。那时候秘书长就是曾维。
  </span>
 </p>
 <p class="p11">
  <span class="s8">
   知情人认为，习近平比毛泽东厉害之处在于，毛泽东是搞政治运动，完了过一阵还能翻案，习近平是政治里边查经济，你翻不了身。钱都给你落实了，你能翻吗？
  </span>
 </p>
 <p class="p15">
  <span class="s8">
   “反腐去不了根儿。在底下这么整也整不净。
  </span>
  <span class="s9">
   先查谁，后查谁，不还他们说了算嘛。”他说。
  </span>
 </p>
 <p class="p12">
  <span class="s10">
   <b>
    他
   </b>
  </span>
  <span class="s1">
   举例说，山西政协主席李佳在二十大之前就给查撤了。查李佳是冲着胡春华，李佳当时从沈阳调内蒙古，胡春华是内蒙古书记，他是秘书长。一查李佳，胡春华基本就完事儿了。
  </span>
 </p>
 <p class="p12">
  <span class="s1">
   中国民主党全国联合总部主席宁先华9月22日也告诉大纪元记者，已确认曾维进去了。同天进去的还有中一集团老总李东齐，即万鑫、万豪、东北国际医院大老板，也是沈阳全运村、市委市政府等办公大楼承建商。李东齐一进去，还会供出更多人。沈阳这场地震不亚于沈阳“慕马”大案。
  </span>
 </p>
